Defined in build/plugins/core/Http/Http.js:30

Make http calls. It supports HTTPS and follows redirects by default.

Parameters:

  • requestConfig Object

    A request configuration made up of key/value pairs. Wraps the popular npm request module so any request config supported by this module will work. See - https://www.npmjs.com/package/request#requestoptions-callback.

Returns:

Promise:

Promise resolved with an object like so { statusCode: 200, body: 'Hello World' }

Example:

var requestConfig = {
                        url: 'http://example.com',
                        qs: { searchString: 'foo', page: 1 },
                        method: 'GET',
                        headers: { 'Content-Type': 'MyContentType', 'Custom-Header': 'Custom Value' }
                       };
ETL.Plugins.Core.Http.makeRequest(requestConfig);
